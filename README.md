== Description ==
LearnPress ReviewX is a course review and rating addon for LearnPress. It allows students to leave reviews and star ratings for courses.

== Features ==
* ⭐ Course rating system (star-based)
* 💬 Student review submission
* 📊 Automatic average rating calculation
* ⚡ Lightweight and performance optimized
* 🔌 Deep LearnPress integration
* 🧠 Developer-friendly and extendable architecture

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/learnpress-reviewx/`
2. Activate the plugin through the WordPress admin panel
3. Ensure LearnPress is installed and active
4. Start collecting course reviews

== Requirements ==
* WordPress 5.8+
* PHP 7.4+
* LearnPress plugin installed

== Frequently Asked Questions ==

= Does it work without LearnPress? =
No, LearnPress is required for this plugin to work.

= Is it customizable? =
Yes, it is built with an extendable OOP structure.
